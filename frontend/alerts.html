📧 EmailJS (Free — 200 emails/month)
  1. Sign up at emailjs.com
  2. Add an Email Service → connect Gmail/Outlook
  3. Create Email Template — use variables:
    {{to_email}} {{subject}} {{message}} {{from_name}}
  4. Copy Service ID, Template ID, Public Key
  5. Open alerts-page.js and paste below the ▼ PASTE HERE comment
const EMAILJS_SERVICE_ID = 'service_xxxxx'; const EMAILJS_TEMPLATE_ID = 'template_xxxxx'; const EMAILJS_PUBLIC_KEY = 'your_public_key';
